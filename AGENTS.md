# Instruções para Inteligências Artificiais e Agentes Autônomos

Bem-vindos ao projeto **Engenharia de Software com IA Aplicada**!

Este arquivo serve como entrada universal (compatível com Cursor, Cline, Windsurf, GitHub Copilot e Antigravity).

## Diretrizes Padrão
1. Sempre que você for questionado sobre a estrutura do repositório ou abrir uma nova sessão, **consulte primeiramente o resumo contido em**: `.agents/skills/repo_context.md`
2. Todas as "skills" (fluxos operacionais, refatorações, testes) e regras de negócio da equipe ficam alocadas no diretório `.agents/skills/`. Atue de acordo com elas se solicitado.
3. Este repositório utiliza **MCP (Model Context Protocol)** e **Agent Skills**. Em implementações focadas em IA, prefira esta abordagem moderna de ferramentas.
